Learn the Basics

Understanding key stock trading concepts

What is a Stock?

A stock represents a unit of ownership in a company. When you buy a stock, you become a partial owner of that business.

In real markets, stock prices change based on company performance, investor expectations, and overall market conditions. In StockEasy, prices come from a live stock market API, so you interact with real market data using virtual money.

Buying Stocks

Buying means purchasing shares at the current market price.

  • The cost is deducted from your cash balance
  • The shares are added to your portfolio
  • You profit if you later sell at a higher price

Example
Buy 10 shares of Apple (AAPL) at $150 each:
Total cost = 10 × 150 = $1,500

Selling Stocks

Selling means selling shares you already own at the current market price.

  • Shares are removed from your portfolio
  • Sale proceeds are added to your cash balance
  • You realize a profit if you sell above your purchase price, or a loss if you sell below it

Example
Sell 10 shares at $160 each:
Cash received = 10 × 160 = $1,600
If you originally paid $1,500, profit = $100

Portfolio

Your portfolio is the set of all stocks you currently own. It typically shows:

  • Quantity of shares for each stock
  • Purchase price (average cost)
  • Current market value

This gives you a snapshot of your overall positions in StockEasy.

Profit & Loss (P&L)

P&L (Profit and Loss) shows how much you have gained or lost on your trades.

  • Unrealized P&L: Profit or loss on positions you still hold, based on the current price
  • Realized P&L: Profit or loss from positions you have already sold

Both help you understand how your active trades and completed trades are performing.

Market Price

The market price is the current trading price of a stock. It changes throughout the day as buyers and sellers interact.

In StockEasy, prices are fetched from a live stock market API, so the values you see closely match current market prices.

Cash Balance

Your cash balance is the amount of virtual money available to trade.

In StockEasy:

  • You start with $100,000 in virtual cash
  • Buying stocks decreases your cash balance
  • Selling stocks increases your cash balance

Your cash balance effectively sets your trading budget in the simulator.

How to Run the App

Follow these steps to set up and run StockEasy on your local machine:

Step 1: Download the Code from GitHub
  1. Go to the StockEasy GitHub repository
  2. Click the Code button and select Download ZIP, or clone using Git:
    git clone https://github.com/username/StockEasy-main.git
  3. Extract the ZIP file to your desired location
Step 2: Install Prerequisites
  • Java Development Kit (JDK) 11 or higher - Download from Oracle
  • Eclipse IDE - Download Eclipse (choose "Eclipse IDE for Enterprise Java and Web Developers")
  • Maven - Typically comes with Eclipse, but can be downloaded from Maven Official Site
Step 3: Open Eclipse and Import as Maven Project
  1. Open Eclipse
  2. Go to File → Import
  3. Select Maven → Existing Maven Projects
  4. Click Next
  5. Browse and select the folder where you extracted the StockEasy project
  6. Click Finish - Eclipse will import the project and download all dependencies automatically
Step 4: Configure the Application
  1. Navigate to: src/main/resources/application.yml
  2. Update the configuration with your environment details (database, API keys if needed)
  3. Save the file
Step 5: Run the Application
  1. In Eclipse, right-click on the project name
  2. Select Run As → Spring Boot App
  3. Wait for the application to start (you should see "Started StockEasyApplication" in the console)
Step 6: Access the Application
  1. Open your web browser
  2. Navigate to: http://localhost:8080
  3. You should see the StockEasy login page
  4. Register a new account or log in to start trading!
Troubleshooting:
If you see "Spring Boot App" option is missing, install the Spring Boot Extension for Eclipse or ensure STS (Spring Tools) is installed.
